# stocks_analyzer_coreとは（Overview）

stocks_analyzerアプリケーション共通機能を提供するPlay Framework 2.5ベースのサブプロジェクトです。<br>
stocks_analyzer_core参照先

* [stocks_analyzer_batch](https://github.com/chotto-martini/stocks_analyzer_batch)

stocks_analyzer_coreが提供する主な機能は、アプリケーション共通ログ出力機能、データベースアクセス機能、データベースモデル定義、google API操作、アプリケーション定数定義、を提供します。



## インストール（Install）

stocks_analyzer_core単体では機能しない為、各アプリケーションを参照ください。

* [stocks_analyzer_batch](https://github.com/chotto-martini/stocks_analyzer_batch)
